# 地质数据系统操作指南

## 1. 系统主页
系统现已重构，主页直接展示**地质数据大厅**。
- 左侧侧边栏：包含智能搜索框、分类过滤器和数据统计。
- 右侧地图：展示所有地质点位（聚类显示）。

## 2. 智能搜索
在侧边栏顶部的搜索框中输入关键字即可进行多字段模糊查询。
支持的查询字段包括：
- **样本编号** (如 Sample-0422)
- **年代** (如 Archean)
- **岩性** (如 Sandstone)
- **构造** (如 Fold)
- **矿产** (如 Coal)
- **海拔** (如 1547.38)
- **采样日期** (如 2019-07-16)

系统会自动高亮显示匹配的字段内容。

## 3. 数据导出
点击侧边栏顶部的“导出数据”按钮，选择所需格式：
- **Excel (.xlsx)**：通用表格格式。
- **CSV (.csv)**：逗号分隔值格式。
- **Shapefile (.zip)**：GIS专业矢量数据格式（包含 .shp, .shx, .dbf 等）。

导出范围为当前筛选条件下的所有数据。

## 4. 地图交互
- **点击聚类点**：自动缩放至该区域。
- **点击单个点**：弹出详细信息卡片，展示地质属性（年代、岩性、构造等）。
